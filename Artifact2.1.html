Artifact 2.1:
Color World

Course: Designing Games for Online Learning

Description

This video provides an overview of an analog game designed to teach color theory.

How it Represents Competency 2.1

This project used game design frameworks to develop the game based on color theory principles.

Reflection

I have never been a person who plays games so this class was a bit of a stretch for me. The main lesson I learned in this course was that you have to play games to get an idea of how to design games. Games have an import role in education so I will seek ways to use them in the future.

Another lesson I learned is that games take a great deal of time to develop, especially an educational game which has to align with learning objectives. Teachers can create games like this with materials found at any big box store. I do wonder what it takes for a game to be successful commercially and if there is a market for analog educational games.
