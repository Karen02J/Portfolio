Reflection

I started this project with a list of learning objectives. Of course my goal is to always include someone the higher levels of Bloomâ€™s Taxonomy. I thought that it would be impossible to represent these levels in a stand-alone course. I was amazed that I found a way to use the higher levels of Blooms and make my course more engaging for users. I am most proud of my interactive color slider.

I also realized that there are a lot of moving parts when creating this type of content. For example, the branching scenarios must go to the correct slide and users must get the appropriate feedback based on the choices they made. It also would be easy to get the users stuck in an endless loop and unable to exit. Thankfully, everything worked for me although this project could benefit from some usability testing.

Artifact Information

The self-paced online tutorial contains:

  • Interactive activities
  • Branching scenarios
  • Feedback based on choices made
  • Checks on learning at the end of each section
  • An end-of-course quiz
