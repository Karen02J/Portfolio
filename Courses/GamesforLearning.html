Designing Games for Learning Fall 2019

Course Description

Instructor: Dr. Danielle Oprean

This course—via hands-on activities and projects—will immerse you into designing analog and digital games for learning. You will learn how games can provide and support meaningful learning outcomes and learn to implement the strategies and design features in games that will enable such learning to take place.

Course Competencies

  • Distinguish between different types of educational games.
  • Design and paper prototype an analog game using sound principles of game design.
  • Analyze video games to identify principles of learning design.
  • Engage in a gaming community in order to inform game design.
  • Create a pedagogically sound lesson plan to use an existing game in an educational manner.
  • Complete a design document for a digital learning game in a domain of your choosing.
  • Utilize primary functions of a video game development shell (e.g. Stencyl).
  • Design assessments for learning games.
