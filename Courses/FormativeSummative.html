Course Description

Instructor: Dr. Isa Jahnke

This course will prepare you to work within the Canvas Learning Management System (LMS) to set up and manage an online course. As a student in this course, you will investigate and manipulate components of Canvas from an instructorâ€™s perspective.

Learning Outcomes

  • Creating Topic/Weekly Modules containing links, files, quizzes, assessments, and using requirements and prerequisites
  • Adding Files, manipulating files, uploading/expanding zip files for learning objects
  • Assessments
  • Creating, Grading, and Evaluating Assessments
  • Understanding Analytics
  • Calendar systems, Teacher scheduler and Appointment Groups
  • Managing Grade Administration
  • Basic Learning Tools Interoperability (LTI) Integrations
  • Using Activity tools for Communication and Collaboration
