Introduction to Digital Media
Summer 2018

Course Description

Instructor: Dr. Krista Galyen

Hands-on approach to multimedia production techniques. Develops understanding of technical and conceptual tools for the basics of digital media, video editing, still image and audio file manipulation. Students create web portfolio to present their digital products.

Course Objectives

  • Create digital files for the Web using relevant software.
  • Understand and apply copyright law to educational content.
  • Upload and download digital files to and from Web servers.
  • Learn to create web pages using accepted design and usability standards.
  • Create digital media effectively targeting specific learning objectives.
