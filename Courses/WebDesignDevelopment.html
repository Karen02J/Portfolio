Web Design & Development
ISLT 7375
Spring 2020

Course Description

Instructor: Dr. Danielle Oprean

Learn how to make websites or customize existing ones using HTML5 and CSS3 (cascading style sheets). This course covers the fundamentals of setting up a website, creating and implementing web design, and exploring responsive web design. Develop a fully functional website ready to use or to allow for moreadvanced web languages to be integrated.

Learning Outcomes

  • Set up an environment for web design and development.
  • Create and validate a proper web page structure.
  • Create and apply cascade style sheets to implement page format and layout.
  • Design a functional website using wireframes and storyboards.
  • Design and implement intuitive website navigation.
  • Implement responsive web design form mobile devices.
  • Expertly use images, audio, and video on web pages.
  • Construct effective web tables and forms.
  • Design and develop a fully functional website.
