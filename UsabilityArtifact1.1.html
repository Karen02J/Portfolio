Usability Artifact 1.1:
Oboe 101

Course: Front End Analysis of Systems

Description

This individual project was created for band directors who teach beginning oboe students. These band directors are likely non-oboe players.

How it Represents Usability Competency 1.1 (Needs Assessment & Task Analysis)

This project is based on the ASK system which helps users to perform job tasks through the interface of a system. This system is designed as though the users are having a conversation with an expert and the types of questions used are job task questions and goal-oriented procedural knowledge questions.

Reflection

This was my first experience with creating interactive PowerPoint slides. For this portfolio, I converted the PowerPoint into a PDF and embedded it into the website. For the first time, I had to think about usability and how the audience could navigate a system. Getting all the landing pages and links to work was quite challenging and, in the end, my project worked as intended. This artifact provides all of the information of a Q and A document with a list of resources, but is more interactive which should help a band director perform the job tasks of teaching beginning oboe students.

Artifact Information

This presentation contains:

  • Buttons to click to find information
  • Information on materials, music, and anything else to get a beginning oboe student started
