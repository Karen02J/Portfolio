Usability Artifact 1.2:
Oboe 101

Course: Front End Analysis of Systems

Description

This individual project is an interactive PDF created for band directors who teach beginning oboe students. These band directors are likely non-oboe players.

How it Represents Usability Competency 1.2

This project is based on the Ask system which helps users to perform job tasks through the interface of a system. This system is designed as though the users are having a conversation with an expert. The types of questions used in this project are job task questions and goal-oriented procedural knowledge questions.

Reflection

This was my first experience with creating interactive Power Point slides. For this portfolio, I made it into an interactive PDF. For the first time, I had to think about usability and how the audience could navigate a system. Getting all the landing pages and links to work was quite challenging and, in the end, my project worked as intended.

Artifact Information

This presentation contains:

  • Buttons to click to find information
  • An embedded video
  • Information on materials, music, and anything else to get a first-year oboe student started
